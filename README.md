# Este Repositório foi criado com o príncipio de revisar tópicos de LPII
## Sobre o curso
- Este curso foi ministrado de forma extra-oficial, sem geração de carga horária ou certificados para os discentes que participaram, e não possui vínculo com quaisquer delegação da universidade, sendo realizado de forma completamente voluntária.
- Este curso não **SUBSTITUI** a matéria de Linguagem de Programação II, servindo apenas para complementar o que foi ensinado e ajudar a solucionar dúvidas.
- O curso foi ministrado nos dias 20/01/26 e 21/01/26 pelo monitor Emanuel Humberto Menezes Cerqueira.

## Guia Rápido
- O curso é composto por 4 partes e um Exercício Final
    - Parte 1: Ponteiros
    - Parte 2: Funções
    - Parte 3: Ponteiros e Funções
    - Parte 4: Aquivos
- O Exercício Final é composto por 5 Questões, as respostas deverão ser enviadas como um repositório no *github*. Não é de caráter obrigatório, porém ajudará na fixação dos conteúdos."# Curso-de-Ferias-LPII"
